Grille de mise en page

"Responsive"

Vous pouvez définir plusieurs lignes à l'intérieur d'un même conteneur .avn-flex. Vous pouvez également adapter le nombre de cellules par lignes en fonction de la largeur disponible à l'écran.

Pour cela, utilisez le pattern suivant : .avn-flex-[ 1-12 ][ -medium|-large ].
L'exemple suivant affiche les cellules :

  • sur 2 colonnes pour les écrans de largeur inférieure à @width-medium.
  • sur une ligne 3 colonnes pour la première ligne et sur 1 colonne pour la suivante, pour les écrans de largeur inférieure à @width-large.
  • sur 4 colonnes, pour les écrans de largeur supérieure à @width-large.
