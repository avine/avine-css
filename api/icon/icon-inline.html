Icônes en ligne

Les icônes sont basés basés sur la librairie FontAwesome.

Pour afficher une icône, commencez par ajouter la classe .avn-icon au conteneur de votre choix. Ensuite, ajoutez la/les classe(s) .avn-icon-before et .avn-icon-after pour afficher les icônes avant ou après le contenu. Enfin, ajouter la classe indiquant le type d'icône attendu, comme par exemple .avn-icon-quotes.

Par défaut, les icônes sont affichés en ligne.

<div class="avn-icon avn-icon-before avn-icon-quotes">
	Lorem ipsum
</div>
<div class="avn-icon avn-icon-after avn-icon-quotes">
	Lorem ipsum
</div>
<div class="avn-icon avn-icon-before avn-icon-after avn-icon-quotes">
	Lorem ipsum
</div>
