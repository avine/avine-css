Icônes en blocs

La configuration des icônes en blocs est la même que pour les icônes en ligne en rajoutant simplement la classe .avn-icon-block.

Les icônes sont positionnés horizontalement lorsque la largeur de l'écran est supérieure à @width-medium. Sinon, ils sont positionés verticalement.

<div class="avn-icon avn-icon-block avn-icon-before avn-icon-quotes">
	Lorem ipsum
</div>
<div class="avn-icon avn-icon-block avn-icon-after avn-icon-quotes">
	Lorem ipsum
</div>
<div class="avn-icon avn-icon-block avn-icon-before avn-icon-after avn-icon-quotes">
	Lorem ipsum
</div>
