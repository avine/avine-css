Régions

"bound > fill"

Comme nous l'avons déjà mentionné, la classe .avn-area-fill ajoute au conteneur le bord supérieur (et éventuellement le bord inférieur avec la classe .avn-area-border).

La classe .avn-area-bound > .avn-area-fill ajoute les bords droit et gauche afin de fermer entièrement le conteneur. Ces bords apparaîssent uniquement lorsque la largeur de l'écran est supérieure à @width-large.

Dans cette configuration, chaque conteneur reçoit automatiquement des marges internes. La classe .avn-area-pad n'est plus nécessaire. Pour retirer cette marge, utilisez la classe .avn-area-nopad.

Lorsque la largeur de l'écran est supérieure à @width-large, le dernier conteneur .avn-area-bound > .avn-area-fill:last-child reçoit automatiquement le bord inférieur. La classe .avn-area-border n'est plus nécessaire, sauf si vous souhaitez ajouter ce bord même pour des largeures inférieures à @width-large.
