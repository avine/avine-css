Régions

Bandeaux

La classe .avn-area-over transforme son conteneur .avn-area-fill en bandeau, lorsque la largeur de l'écran est supérieure à @width-large. Concrètement, la largeur du conteneur déborde sur l'axe horizontal au-délà de @width-large pour obtenir l'effet bandeau.

Pour orienter le bandeau vers le haut, ajoutez la classe .avn-area-over-top.
Pour orienter le bandeau vers le bas, ajoutez la classe .avn-area-over-bottom.
